## Opus Magnum AI Scheming
# Introduction
The goal of this project is to build an AI model that controls the processes required to make deliberate schemes - plans focusing on successfully fulfilling specific goals. The goals the model can plan to achieve are somewhat limited in scope. In short, we only consider goals that pertain to important areas. We do not want to build an AI model that helps make decisions about whether to buy this or that cone of ice cream, for instance.

Another important point to stress is that our AI model should be firmly based on practical experience. This does not mean we cannot allow for the conception of theoretical plans, but they must ultimately be feasible in the real world.
