## Opus Magnum AI Scheming
# Introduction
The goal of this project is to build an AI model that controls the processes required to make deliberate schemes (plans focusing on successfully fulfilling specific goals). The goals the model can plan to achieve are somewhat limited in scope. In short, we only consider goals that pertain to significant domains. For instance, we do not want to build an AI model that helps make trivial decisions, such as which cone of ice cream to buy.

Another important point to stress is that our AI model should be firmly based on practical experience. This does not mean we cannot allow for the conception of theoretical plans, but they must ultimately be feasible in the real world.
